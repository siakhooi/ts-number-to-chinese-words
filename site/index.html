Install

The library number-to-chinese-words can be used in 2 ways:

  1. npm
  2. cdn

Repo: https://github.com/siakhooi/ts-number-to-chinese-words/

npm

Install: 
        $ npm i @siakhooi/number-to-chinese-words
Then in javascript file: 
        const NumberToChineseWords = require('@siakhooi/number-to-chinese-words');
        console.log(NumberToChineseWords.convertNumber(7));

cdn (browser)

Use one of these

<script src="https://siakhooi.github.io/ts-number-to-chinese-words/number-to-chinese-words.bundle.js"></script>

<script src="https://unpkg.com/@siakhooi/number-to-chinese-words/dist/number-to-chinese-words.bundle.js"></script>

<script src="https://cdn.jsdelivr.net/npm/@siakhooi/number-to-chinese-words/dist/number-to-chinese-words.bundle.js"></script>

          <HTML>
