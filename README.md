## Features

- Translate numbers into Chinese characters for:
  - Integer -9999,9999,9999 to +9999,9999,9999
  - Simplified Chinese

## Usage

```javascript
const NumberToChinese = require('@siakhooi/number-to-chinese-words');
console.log(NumberToChinese.convertNumber(7));
```
